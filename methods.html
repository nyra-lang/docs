Built-in methods

One-page lookup for compiler builtins and stdlib method syntax on strings, numbers, arrays, vectors, maps, and objects. No import required for language builtins; import paths shown where needed.

Quick tip

Press Ctrl+K and type a method name (e.g. split, push, abs). For full module APIs (HTTP, crypto, FS), see Standard library.

.ends_with(suffix)stringi32Suffix test
.replace(from, to)2 × stringstringReplace all occurrences (Rust-style)
.replacen(from, to, count)2 × string, i32stringReplace at most count occurrences (1 = first only)
.to_upper()—stringASCII upper-case
.to_lower()—stringASCII lower-case
clone s—stringHeap copy (keyword, not a method)

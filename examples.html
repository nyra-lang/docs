1. Hello World beginner

One file · no imports · ~5 lines

hello/ main.ny
main.nyfn main() {
    print("Hello, world!")
}
  1. Create folder hello and save main.ny.
  2. Run: nyra run main.ny (single file is OK here).

Output: Hello, world!

2. Project scaffold tooling

NyraPkg · auto main.ny

mkdir myapp && cd myapp
nyra pkg init
nyra run .

Creates nyra.mod, nyra.lock, nyra.sum, and main.ny (Hello World).

3. Multi-file greet app imports

Import functions from src/ · must use nyra run .

greet-app/ main.ny nyra.mod src/ hello.ny
src/hello.ny
fn hello() {
    print("Hello from src/hello.ny")
}

fn hello_twice() {
    hello()
    hello()
}
src/hello.ny
fn hello() {
    print("Hello from src/hello.ny")
}

fn hello_twice() {
    hello()
    hello()
}
main.ny
import "src/hello.ny"

fn main() {
    hello()
    hello_twice()
}
main.ny
import "src/hello.ny"

fn main() -> void {
    hello()
    hello_twice()
}
  1. nyra pkg init in greet-app (or create files manually).
  2. Add src/hello.ny and update main.ny as above.
  3. From project root: nyra run .
Imports guide See Imports guide for functions, structs, and constants.

4. Sum loop loops

for .. range · prints 30

main.ny
fn main() {
    let mut sum = 0
    for i in 0..10 {
        sum = sum + i
    }
    print(sum)
}
main.ny
fn main() -> void {
    let mut sum: i32 = 0
    for i in 0..10 {
        sum = sum + i
    }
    print(sum)
}
nyra run main.ny

Output: 30 (sum of 0..9)

5. Calculator structs + imports

Official repo example · three files

calculator/ main.ny types.ny ops.ny
types.nystruct Calculator {
    value: i32
}
ops.ny
import "types.ny"

fn add_calc(c, n) {
    return Calculator { value: c.value + n }
}
ops.ny
import "types.ny"

fn add_calc(c: Calculator, n: i32) -> Calculator {
    return Calculator { value: c.value + n }
}
main.ny
import "types.ny"
import "ops.ny"

fn main() {
    let c = Calculator { value: 0 }
    let c2 = add_calc(c, 10)
    print(c2.value)
}
main.ny
import "types.ny"
import "ops.ny"

fn main() -> void {
    let c = Calculator { value: 0 }
    let c2 = add_calc(c, 10)
    print(c2.value)
}
git clone git@github.com:nyra-lang/nyra.git
cd Nyra
nyra run examples/projects/calculator

Output: 10

7. Enum & match pattern matching

Single file · tagged enums

main.ny
enum Color {
    Red
    Green
    Blue
}

fn main() {
    let c = Color.Red
    let n = match c {
        Color.Red => 1
        Color.Green => 2
        Color.Blue => 3
    }
    print(n)
}
main.ny
enum Color {
    Red
    Green
    Blue
}

fn main() -> void {
    let c = Color.Red
    let n = match c {
        Color.Red => 1
        Color.Green => 2

    }

}

8. Dungeon Steps reference app

Official app at App/NyraApp/ · multi-file · const, enum, match, &&, %

The largest beginner-friendly sample in the repo: a small dungeon simulation split across config, types, world, and engine modules. Ideal after you understand single-file examples and the calculator project.

App/NyraApp/ main.ny src/ config.ny types.ny world.ny engine.ny
cd App/NyraApp
nyra run .

Output (first lines): Dungeon Steps, then 65, 0, 3

12. C library: zlib (C Bindgen) FFI

System C library · auto extern fn · no hand-written FFI

Nyra calls real native code (zlib, SQLite, OpenSSL, …) with one bind command. Full walkthrough: C Bindgen → zlib tutorial.

Commands

nyra pkg init && cd myapp
nyra pkg c add zlib
nyra run .
myapp/ main.ny nyra.mod vendor/bindings/ zlib.ny
import "vendor/bindings/zlib.ny"

fn main() {
    print("Hello, World!")
    print(zlibVersion())
}
import "vendor/bindings/zlib.ny"

fn main() -> void {
    print("Hello, World!")
    print(zlibVersion())
}

Output:

Hello, World!
1.3.2

See also: SQLite tutorial · Troubleshooting · examples/c_bindgen/ (custom C + link-source)

13. Raylib graphics (C Bindgen) FFI

Game window · draw loop · shapes & text · native GPU speed

Full walkthrough: C Bindgen → Raylib. The header is at /opt/homebrew/opt/raylib/include/raylib.h (not /opt/homebrew/include/).

Commands

nyra pkg c add raylib
nyra run .

Or: cd examples/c_raylib && nyra run .

Or run the bundled example (after brew install raylib):

cd examples/c_raylib
nyra run .

Opens an 800×450 window with white background, red text, and a red circle at 60 FPS.
